Quick Reference

What is MCP
Model Context Protocol. One shared standard for Claude to call real tools — Drive, Slack, a database — instead of a one-off integration per tool.
Why sample-first
A live connector can fail mid-room. A sample path proves the same behavior with zero dependencies, every time.
Talk track
"It's not guessing the category from the filename — it's reasoning over what the tool actually returned."

Privacy

Nothing run here is stored or analyzed afterward. This lab exists for one purpose: organizing your Drive assets by category and tag — in the open. The signal trace below shows exactly what was sent and returned, every time. Personal stays personal.
